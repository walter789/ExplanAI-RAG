# ExplanAI-RAG

Production-ready ML Model Debugging & Explainability Assistant.

## Features
- **MLflow Integration:** ETL pipeline for experiment metrics and parameters.
- **Vector Storage:** Qdrant-backed semantic search for experiment facts.
- **Interpretability Agent:** Grounded RAG system for root-cause analysis.
- **Systematic Evaluation:** Ragas-based performance benchmarking.
- **Streamlit Dashboard:** Intuitive UI for experiment interrogation.

## Setup
1. Clone the repository.
2. Create a virtual environment: `python -m venv venv`.
3. Activate the environment and install dependencies: `pip install -r requirements.txt`.
4. Run the dashboard: `streamlit run app.py`.
